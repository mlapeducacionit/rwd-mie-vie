Tipos de Diseño

Maquetación de un sitio

Desde que existen las páginas web y con la aparición de CSS en el año 90 surgieron diferentes formas de estilizarlas, utilizando medidas absolutas, relativas e incluso flexibles y de todos los métodos implementados podemos destacar principalmente 3, que no necesariamente se aplican de manera individual:

  • Fluid Web Design Estilo que porcentajes para mantener las proporciones de los elementos de la página con respecto a sus diseño. Debido a que solo utilizan porcentajes, estos diseño no suelen ser escalables en pantallas pequeñas y grandes. Una solución a este problema surge con el Liquid Web Design que cosiste en la implementación de máximos y mínimos pero en pantallas grandes se generan espacios no contemplados
  • Adative Web Design Estilo que consiste en crear una página para cada dispontivo objetivo, genertalmente smarphone, table, desktop, es bastante practico cuando contemplamos disponistivos de bajos recursos, como poca capacidad de procesamiento o ancho de banda limitado, pero impllica tener cotenido duplicado. Es fácil de mantener ya que maneja archivos por separado y es una solución practica par aclientes con bajo presupuesto, además suelen implementarse diferentes estilos según el tamaño de la pantalla, empezando generamente desde la vista de escritorio y reduciendo los contenidos para los otras vistas.
  • Responsive Web Design Estilo sugerido originalmente por Ethan Marcotte en una publicación del sitio A list apart donde aplica un concepto como One Web for All donde habla del uso de media-queries sistemas de grillas con porcentajes y multimedial flexible entre otros conceptos. Se aconsjea implementar una caracteristica conocida como Content-first evolución del Mobile-first, que consiste en planificar el diseño desde la vista móvil utilizando los contenidos finales para tener una clara noción de como se vera la página en el navegador, para luego ir cambiando el diseño según las dimensiones de la pantalla donde se reproduzca.

Unidades de medidas

Usos y aplicaciones

  • Absolutas Son aquelas que representan siempre el mismo valor independemente de la pantall del dispositivo donde se visualice la página Se utilizan principalmente para establcer los límites de tamaño de los elementos de la página web, tanto máximos como mínimos, respecto al ancho de los mismos.
    • distancia: mm, cm, in(2.54cm)
    • tipografia:pc(1/6in), pt(1/72in 0.35mm)
    • hidridas:px (1/96in) su representaión depende la resolución de la pantalla
  • Relativas Son aquellas que dependen de otro elmento, generalmente el contenedor, para establecer su media; por lotanto un valor no siempre representa lo mismo. Se aplicacn principalmente a la tipografía, atributos relacionados con esta como margin, padding y border, e incluso para definitir el tamaño de las cajas de cotenido
    • tamaño: %(tamaño), fr(fracción)
    • tripografía: em(ancho "M"), ex(alto de "x"), ch(ancho de "0"), %(letra)
    • hidrida: rem (root em) su unica referencia es la etiqueta raíz(html)
  • Flexibles Son aquellas que toman como referente las dimensiones del viewport (ventana del navegador) y a diferencia del resto, no dependne de ningún otro contenedor. Se implementan principalmente para la generación de tipografía responsive, establer tamaños en vertical debido a que por defecto ningún elemento posee una altura definida.
    • dimensión:vw(% ancho del viewport), vh (% alto viewport)
    • proporción:vmin (% menor medida), vmax(% mayor medida)

Bootstrap

Framework CSS

Conjunto de herramientas desarrolladas por el equipo de twitter implementado sass y es6 (en las últimas versiones), en versiones anteriores teníamos jQuery. Cuya finalidad es facilitar la creación de sitios web responseive, accediendo a los estilos y scripts provistos por los desarrolladores del framework

Podemos implementarlo de manera externa en nuestro proyecto a través de la vinculación del CDN (content distribution network) o descargar los archivos fuentes, ya sea la hoja de estilos compliada o los archivos de procesador.

Estilo de utilidad o ayuda

Una de las caracteristica de Bootstrap radica en sus helpers o utilitarios que son basicamente clases CSS que se utilizan dentro de la estructra para definir la apariencia básica de los elementos. La estructura de dichas clases suele ser la siguiente atributo-valor(unidad) reprentando el atributo con una letra o la palabra completa dependendiendo de que tan general sea este

A partir de la verisón 4.x como el enfoque de boostrap cambió para compatibilizar los sitios con los diferentes tipos de escritura (originalmente de texto) las clases cuyo valor corresponde a una orientación como izquierda y derecha ya no reciben esos nombre,s sino que pasan a llarmase start y end

  • margin
    • margin: .25rem; .m-1
    • margin-left: 1rem; .ms-3
    • margin-top: .5rem;
    • margin-right: 1.5rem; margin-left: 1.5rem; .mx-4
    • margin-bottom: 3rem; .mb-5
    • margin-top: 1rem; margin-bottom: 1rem; my-3
  • padding
    • padding: .25rem; .p-1
    • padding-left: 1rem; .ps-3
    • padding-top: .5rem; .pt-2
    • padding-right: 1.5rem; padding-left: 1.5rem; .px-4
    • padding-bottom: 3rem; .pb-5
    • padding-top: 1rem; padding-bottom: 1rem; py-3
  • border
    • border: 3px solid; .border .border-3 .border-primary
    • border-left: 1px solid; .border-start
    • border-top: 4px solid; .border-top .border-4
    • border: 1px solid; border-radius: .25rem; .border .rounded
    • border: 5px solid; border-radius: 50rem; .border .border-5 .rounded-pill .border-primary
    • border-radius: 50%; .border .rounded-circle

Componentes y sistema de grillas

Bootstrap aplica la metodología BEM (Block Element Modifier) para nombrar a las clases que forman parte de su sistema, entre las cuales encontramos a los componentes. Estos son, básicamente, clases que deben implementarse en el HTML de cierta manera para generar diferentes estilos visuales predeterminados, como por ejemplo navegación, galerías, modales, tablas, etc...

Esta metodología se rige por las siguientes reglas:

  • El contenedor (bloque) define el nombre principal para los demás elementos y así distinguir su jerarquia en los estilos, por ejemplo: .nav y .nav-link

Bootstrap se caracteriza por su sistema de grillas de 12 columnas, el cual consiste en un utilizar un contedor flex y asignar a cada elemento un número de columna correspondiente al tamaño a ocupar.

El calculo que se realiza par adeterminar el tamaño de cada columna es el siguiente Nº columna / total columnas como por ejemplo col-4 ( 4 / 12 ) = 33.33%. Debemos tener en cuenta que la sumatoria de columna no debe superar 12 para que los elementos sigan en la misma fila.
